发表于 2022-03-18 更新于 2022-03-19 分类于 ROS

Ubuntu安装与配置

安装

  • 使用树莓派镜像烧录程序

  • 选择 other general purpose OS

  • 选择Ubuntu

  • 选择Ubuntu Server 20.04.4，注意是64bit版本

  • 配置系统

    账户：pi
    密码：pi

配置

  • 配置WiFi

    烧录成功后打开sd卡中的system-boot，找到network-config，若是没有刚刚配置的wifi，手动添加WiFi信息

  • 换清华源，这里是arm架构的源

    sudo nano /etc/apt/sources.list

    更换完后更新列表

    sudo apt update
